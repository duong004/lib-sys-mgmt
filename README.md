# MVP.Library Management System

## 1. Giới thiệu
**MVP.Library Management System** là một ứng dụng mô phỏng hệ thống quản lý thư viện.  
Sản phẩm giúp quản lý sách, độc giả và hoạt động mượn/trả một cách đơn giản, trực quan.  
Đây là đề tài mang tính thực tế, dễ hiểu, phù hợp cho việc học tập và thực hành lập trình hướng đối tượng.

---

## 2. Ý nghĩa & Tác dụng
- **Ứng dụng thực tế**: Mô phỏng hoạt động của thư viện, giúp người dùng hình dung rõ quy trình quản lý.
- **Học tập & Thực hành**: Là môi trường để áp dụng kiến thức OOP, xử lý dữ liệu, và quản lý hệ thống.
- **Phát triển kỹ năng**: Giúp sinh viên rèn luyện tư duy phân tích, thiết kế, và triển khai phần mềm.
- **Quản lý hiệu quả**: Hỗ trợ thủ thư và độc giả trong việc mượn/trả sách, theo dõi lịch sử, và thống kê.

---

## 2. Tính năng chính
### Quản lý Sách
- Thêm sách mới
- Cập nhật thông tin sách
- Xóa sách
- Tìm kiếm sách (theo tiêu đề, tác giả, ISBN, thể loại)
- Hiển thị danh sách sách

### Quản lý Độc giả
- Đăng ký độc giả mới
- Cập nhật thông tin độc giả
- Xóa độc giả
- Tìm kiếm độc giả

### Quản lý Mượn/Trả
- Mượn sách
- Trả sách
- Gia hạn sách
- Kiểm tra hạn trả
- Tính phí phạt (nếu trễ hạn)
- Xem lịch sử mượn sách

### Báo cáo & Thống kê
- Top sách được mượn nhiều nhất
- Top độc giả tích cực nhất
- Danh sách sách quá hạn
- Thống kê theo tháng
---

## 4. Cách chạy project
1. **Clone project** về máy:
   ```bash
   git clone https://github.com/username/library-management-system.git
   ```
2. Mở project bằng IntelliJ IDEA (hoặc Eclipse/VS Code).

3. Chạy file Main.java để khởi động chương trình.

4. Sử dụng menu hiển thị trên màn hình để thao tác với hệ thống:

```
=== LIBRARY MANAGEMENT SYSTEM ===
1. Quản lý Sách
2. Quản lý Độc giả
3. Mượn/Trả sách
4. Báo cáo & Thống kê
5. Thoát
```
